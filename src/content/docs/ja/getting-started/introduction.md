Appy Stamp JavaScript SDKは、任意のウェブサイトでロイヤルティプログラムデータにプログラムからアクセスできるようにします。カスタムロイヤルティランディングページ、ヘッドレスストアフロント、独自のフロントエンドへのスタンプやリワードデータの統合に使用できます。

SDKは依存関係のない軽量なスタンドアロンスクリプト（3KB未満）です。Shopifyに限らず、任意のプラットフォームで動作します。

## できること

**顧客ログインなし（パブリックデータ）:**
- ショップ設定の取得（プログラム名、スタンプブランディング、通貨）
- 獲得ルールの一覧（顧客がスタンプを獲得する方法）
- 利用可能なリワードの一覧
- VIPティア定義の一覧
- アクティブなビーコンの一覧

**顧客ログインあり（認証済み）:**
- 顧客のスタンプ残高、VIPティア、次のリワードの取得
- アクティビティ履歴の閲覧（ページネーション付き）
- 獲得済みリワードコードの一覧
- リワードの交換とディスカウントコードの取得
- アクティビティのトリガー（SNSフォロー、レビュー）
- 生年月日の更新

## 仕組み

1. ページに設定オブジェクトとスクリプトタグを追加
2. SDKが読み込まれ、ショップ設定を取得し、オプションで顧客を認証
3. `window.appyStamp`のメソッドを呼び出してデータ取得やアクションを実行
4. すべてのメソッドはPromiseを返す
